# TechPanda

A personal portfolio and blog site built with Astro and TailwindCSS.

## Features

- 🚀 Built with [Astro](https://astro.build/)
- 💅 Styled with [TailwindCSS](https://tailwindcss.com/)
- 🌙 Dark mode support
- 📱 Responsive design
- 🎨 Clean and minimal UI
- 📝 Portfolio powered by microCMS
